# 🎮 CodeAlpha Hangman Game

A simple command-line Hangman game developed in Python as part of the **CodeAlpha Python Programming Internship**.

## 📌 Project Description

Hangman is a classic word-guessing game where the player tries to guess a hidden word one letter at a time. The player has a limited number of incorrect guesses before the game ends.

## ✨ Features

- Random word selection
- Letter-by-letter guessing
- Displays correctly guessed letters
- Tracks incorrect guesses
- Maximum of 6 wrong attempts
- Win and Lose conditions
- Beginner-friendly Python code

## 🛠️ Technologies Used

- Python 3
- Random Module

## ▶️ How to Run

1. Make sure Python 3 is installed.
2. Open a terminal in the project folder.
3. Run the following command:

```bash
python CodeAlpha_HangmanGame.py
```

## 🎮 Gameplay

- A random word is selected.
- Guess one letter at a time.
- Correct guesses reveal the word.
- Incorrect guesses reduce the remaining attempts.
- Guess the entire word before running out of attempts to win.
